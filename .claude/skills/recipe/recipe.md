# Recipe Skill

다음 단계를 순서대로 실행하세요:

## 1단계: 재료 파일 읽기

`/Users/ahnsoeun/Downloads/AFM-Bootcamp/week-4/homework/ingredients/` 폴더 안의 모든 `.json` 파일을 읽으세요.
- `ls` 명령으로 파일 목록을 먼저 확인하세요
- 각 JSON 파일을 Read 도구로 읽어 재료 정보를 수집하세요
- 수집한 재료: name(한국어), quantity, unit, category, expiryDays를 정리하세요

## 2단계: 레시피 생성

읽어온 재료 목록을 바탕으로 만들 수 있는 레시피 **2~3가지**를 창의적으로 만드세요.

각 레시피는 다음을 포함해야 합니다:
- 요리 이름 (한국어)
- 난이도 (쉬움 / 보통 / 어려움)
- 조리 시간 (분)
- 필요한 재료 목록 (냉장고에 있는 것 위주)
- 단계별 조리법 (번호 목록)
- 팁 또는 변형 아이디어

## 3단계: 마크다운 파일 저장

생성한 레시피를 `/Users/ahnsoeun/Downloads/AFM-Bootcamp/week-4/homework/recipes/` 폴더에 저장하세요.

파일명 규칙: `recipe_YYYY-MM-DD.md` (오늘 날짜 사용)

마크다운 파일 형식:
```
# 오늘의 냉장고 레시피 🍳

> 생성일: YYYY-MM-DD

## 냉장고 재료 현황

| 재료 | 수량 | 유통기한(일) |
|------|------|------------|
| ... | ... | ... |

---

## 레시피 1: [요리 이름]

**난이도:** ⭐⭐  
**조리 시간:** XX분

### 재료
- ...

### 조리법
1. ...

### 팁
> ...

---

## 레시피 2: [요리 이름]
...
```

## 4단계: 완료 보고

저장된 파일 경로를 사용자에게 알려주고, 만들어진 레시피 이름 목록을 간단히 요약해 주세요.
