# Recipe Skill

다음 단계를 순서대로 실행하세요:

## 1단계: 재료 파일 읽기 (필수 — 가장 먼저 실행)

**반드시 레시피 생성 전에 아래 폴더의 JSON 파일들을 전부 읽어야 합니다.**

`/Users/ahnsoeun/Downloads/AFM-Bootcamp/week-4/homework/ingredients/` 폴더에서:
1. `ls` 명령으로 파일 목록을 확인하세요
2. 목록에 있는 **모든 `.json` 파일**을 Read 도구로 하나씩 읽으세요
3. 각 파일에서 name(한국어), quantity, unit, category, expiryDays를 수집하세요
4. 수집이 끝난 후에만 2단계로 넘어가세요

## 2단계: 레시피 생성

읽어온 재료 목록을 바탕으로 레시피 **2~3가지**를 만드세요.

**필수 조건 (모든 레시피에 적용):**
- 1인분 기준으로 재료 양을 조절하세요
- 조리 시간 15분 이내로 만들 수 있는 요리만 선택하세요
- 자취생 난이도 — 칼질 최소화, 특수 도구 불필요, 후라이팬 하나로 가능한 요리 우선
- 기본 양념(간장, 소금, 참기름, 식용유)은 있다고 가정하세요

각 레시피는 다음을 포함해야 합니다:
- 요리 이름 (한국어)
- 난이도 (쉬움 / 보통 / 어려움)
- 조리 시간 (분)
- 필요한 재료 목록 + 1인분 기준 양
- 단계별 조리법 (번호 목록, 각 단계는 1~2문장으로 간결하게)
- 자취생 팁 (설거지 줄이는 법, 남은 재료 활용법 등)

## 3단계: 마크다운 파일 저장

생성한 레시피를 `/Users/ahnsoeun/Downloads/AFM-Bootcamp/week-4/homework/recipes/` 폴더에 저장하세요.

파일명 규칙: `recipe_YYYY-MM-DD.md` (오늘 날짜 사용)

마크다운 파일 형식:
```
# 오늘의 냉장고 레시피 🍳

> 생성일: YYYY-MM-DD

## 냉장고 재료 현황

| 재료 | 수량 | 유통기한(일) |
|------|------|------------|
| ... | ... | ... |

---

## 레시피 1: [요리 이름]

**난이도:** ⭐⭐  
**조리 시간:** XX분

### 재료
- ...

### 조리법
1. ...

### 팁
> ...

---

## 레시피 2: [요리 이름]
...
```

## 4단계: 완료 보고

저장된 파일 경로를 사용자에게 알려주고, 만들어진 레시피 이름 목록을 간단히 요약해 주세요.
