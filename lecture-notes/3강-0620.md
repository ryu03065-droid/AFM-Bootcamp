## Claude Code 세션
- Claude Code에서 대화 하나하나가 하나의 **세션**이다.
- `/resume` 명령으로 이전에 하던 세션을 이어서 불러올 수 있다.
- 실행 시 사용할 버전을 선택하는 UI가 있음.
- `claude --dangerously-skip-permissions` — 매 작업마다 뜨는 승인(yes) 확인을 건너뛰고 Claude Code가 자율적으로 작업하게 하는 옵션.

## Playwright MCP
- "playwright mcp 설치되어 있는지 확인하고, 안 되어 있으면 설치해줘" 같은 요청으로 확인/설치 가능.
- Claude Code가 결과를 스스로 **검증**하는 데 사용됨.
- 실제 브라우저(Chromium)를 직접 띄우고 조작하는 도구 모음:
  - 웹페이지 열기, 클릭, 입력, 스크롤 등 사용자 조작 시뮬레이션
  - 페이지 스크린샷 촬영
  - 페이지 DOM/텍스트 콘텐츠 읽기
  - 콘솔 로그, 네트워크 요청 확인

## 웹 서버 실습
- Node.js로 작성된 서버 파일이 있어야 하고, Claude Code에게 "OO 폴더를 Node.js 서버로 띄워줘"라고 요청하는 방식으로 실습.
  - 예: "webserver-01 폴더를 nodejs 서버로 띄워줘"
- 터미널을 끄면 그 서버(예: 3000번 포트)도 함께 종료된다.
- `server.js`를 생성해주는 전용 에이전트를 새로 받음.
- **스킬과 에이전트는 모두 `.md` 파일**로 정의된다.
